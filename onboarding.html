메타 세팅법
몰바니 기준 메타 광고 세팅 단계별 안내예요.
1. 캠페인단
캠페인명 규칙: [날짜_목표_랜딩_기획전명]
ex. 25.09.23_트래픽_지그재그_단독기획전

예산 방식:
· 캠페인단 예산 → 머신러닝 자동 배분 (테스트 시 사용)
· 세트단 예산 → 세트별 확실히 구분할 때
몰바니는 자유도가 높아서 그때그때 판단해서 설정해요!
2. 세트단
타겟: 기존타겟 / 신규타겟 / 유사타겟으로 구성
· 기존 타겟: 저장된 타겟 말고 새로 직접 입력
· 노출 위치: 모바일 + 인스타그램만
· 예산: 총예산으로 세팅, 대시보드 금액 그대로 정산
3. 광고단
광고명: 자유. 보고서용 00 넘버링 권장
소재 형식: 단일 이미지 / 슬라이드 / 컬렉션
대표 계정: 몰바니 계정으로 설정
세팅 예시 (광고단)
캠페인 구조(신규타겟/기존타겟)와 광고 미리보기 화면 예시예요. 클릭하면 크게 볼 수 있어요.
